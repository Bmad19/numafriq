# Rappels — brouillon de mémo ou note argumentative

À utiliser lorsque le mode « mémo / argumentaire » est actif.

## Structure indicative (adaptable au dossier)

1. Synthèse en une ligne (objet litige ou enjeu).  
2. Faits présentés de manière neutre puis perspective juridiquement valorisée.  
3. Problème de droit articulé (qualifications, régime juridique).  
4. Arguments principal et subsidiaires ; anticipe contre-argumentation.  
5. Demandes formulées précisément.  
6. Liste des pièces / références **à croiser**.

## Principes non négociables

Ne pas affirmer de numérotation d’article, de disposition OHADA précise ou de jurisprudence sans **vérifération sur source officielle** ou dossier utilisateur ; utiliser **[à vérifier]** systématiquement en cas de doute.

## OHADA — prudence

L’ensemble OHADA doit être consulté sur les **sites institutionnels officiels**. La base téléchargée hors sources officielles peut être périmée ; ne jamais traiter cette synthèse comme un code consolidé garanti à jour.
