## Dossiers complémentaires (extrait Markdown)

Ajoutez ici des fichiers **`.md`** (extraits, grilles méthodo, mémos internes ou **reprises courtes** d’articles de lois / AU OHADA après vérification juridique des droits de reproduction).

- Préfixez avec un numéro pour l’ordre de concaténation : `10-mon-memo-au-suretes.md`
- Ne placez pas d’URLs brutes volumineuses de scraping ; téléchargez plutôt les **documents officiels** depuis les instances compétentes (OHADA.org, États membres, etc.), puis créez vos **résumés** en Markdown sous cette arborescence.
- Les **PDF originaux complets** ne sont pas suivis dans le dépôt par défaut (`.gitignore`) : conservez‑les hors Git ou adaptez la politique d’entreprise.

L’assistant bureau lit **ce dossier et** `kb/*.md`.
